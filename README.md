# ys_play

Flutter萤石云直播插件，支持Android和IOS

# 支持
1.账号对接（授权登录）
2.直播(可设置直播分辨率)
3.回放
4.直播、回放边播边录
5.直播、回放边播边截屏
6.云台控制
7.配网
8.对讲(包含半双工对讲和全双工对讲)
